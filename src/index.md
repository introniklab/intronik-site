# IntronikLab: Проектирование безбарьерных систем

Я — инженер цифровой и физической доступности. ![Сергей Мильчаков](/images/ya.jpg) Я помогаю компаниям и частным лицам делать сложные системы понятными для каждого.

## Мои услуги:
- **Промышленная автоматизация:** Консультирование по созданию доступных интерфейсов
- **Умные дома:** Проектирование автоматизированных систем для людей с ОВЗ
- **Рабочие места:** Тестирование софта и оборудования под ключ
- **Доступный интернет:** Тестирование сайтов на корректную работу с программами экранного доступа
- **Шрифт брайля:** Помощь и консультация при изготовлении информационных материаллов
- **Информационная деятельность:** Проведение обучающих семинаров по взаимодействию с людьми с овз
